About WMTS

The Wildlife Monitoring and Tracking System (WMTS) was inspired by my passion for wildlife conservation and my background in software engineering. As a Biology conservation student and a full-stack software engineer at ALX, I saw an opportunity to bridge the gap between technology and conservation efforts. This project is a culmination of my efforts to create a tool that empowers researchers and conservationists to monitor and track wildlife populations effectively. It is a Portfolio Project for Holberton School.
